# Web CSGO Game

## Introduction
Welcome to the **Web CSGO Game**! This is a web-based adaptation of the popular first-person shooter game Counter-Strike: Global Offensive, designed for fun and competitive multiplayer experiences.

## Features
- **Multiplayer Mode**: Play against your friends or join random matches online.
- **User Authentication**: Secure login and registration functionality.
- **Custom Maps**: Explore various custom maps created by the community.
- **Weapon Selection**: Choose from a variety of weapons to customize your gameplay.
- **Game Modes**: Includes Classic, Deathmatch, and Custom modes.
- **Leaderboard**: Track scores and rankings for competitive gameplay.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/22243dsw/web-csgo-game.git
   ```
2. Navigate to the project directory:
   ```bash
   cd web-csgo-game
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```

## How to Play
- After launching the game, you can create an account or log in if you already have one.
- Choose a game mode and join a server to start playing.
- Use the WASD keys to move, and the mouse to aim and shoot.
